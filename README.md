## About

Diary Confessions was one of the very first things I built with code — a small HTML/CSS project from my 2nd year of B.Tech when I was still figuring out how web pages even worked.

It's raw, unpolished, and exactly what beginning looks like.

I'm keeping it here because every developer has a starting point. This is mine.

---

## What I was learning at the time

- Basic HTML structure and tags
- CSS styling and layout fundamentals
- How to put something on a screen using just a text editor

---

## Reflection

Looking back, this project taught me that the only way to get better is to build something — even when you don't fully know what you're doing yet. The gap between this and my later projects like TrackTact Dashboard and Serenity is the proof.
